# Notification services

Argocd supports several notification services:

- Alertmanager
- AWS SQS
- Email
- GitHub
- Google Chat
- Grafana
- Mattermost
- NewRelic
- Opsgenie
- Overview
- PagerDuty
- PagerDuty V2
- Pushover
- Rocket.Chat
- Slack
- Teams
- Telegram
- Webex Teams
- Webhook
